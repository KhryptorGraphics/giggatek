API Integration Example

Mock API Server

This example uses a mock API server for demonstration purposes. To use this example:

  1. Open a terminal in the frontend/mock-api directory
  2. Run npm install to install dependencies
  3. Run npm start to start the mock API server
  4. The server will run on http://localhost:3000

Alternatively, you can run the setup-mock-api.bat or setup-mock-api.ps1 script in the frontend directory.
